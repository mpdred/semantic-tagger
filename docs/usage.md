```
  -filePath string
        a filePath that contains the version number (e.g. "setup.py")
  -filePath-version-pattern string
        the pattern expected for the filePath version (e.g. "version='%s',") (default "%s")
  -git-tag
        if set, create an annotated tag
  -increment string
        if set, increment the version scope: auto | major | minor | patch
  -prefix string
        if set, append the prefix to the version number (e.g. "api-0.1.0")
  -push
        if set, push the created/updated object(s)
  -suffix string
        if set, append the suffix to the version number (e.g. "0.1.0-rc")
  -version string
        if set, use the user-provided version
```
